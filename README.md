# musante.mu

Personal domain landing page for `musante.mu`, a private family domain used exclusively for personalized email addresses.

## About

This repository hosts the static landing page for [musante.mu](https://musante.mu). The domain is not a public service or product — it exists solely to support custom email addresses for the Musante family (e.g. `name@musante.mu`).

The page is intentionally minimal: a single static HTML file with no frameworks, no backend, and no dynamic content.

## Stack

- Plain HTML, CSS, and JavaScript
- Google Fonts
- Hosted on GitHub Pages
